# Final-project-API
## Movhex Project - Algorithms and Data Structures 2024/2025

This repository contains my C implementation of the "Movhex" project, developed as the Final Exam for the Algorithms and Data Structures course at Politecnico di Milano (Grade: 27/30).

## Project Description

The program simulates a calculation system for optimal routes of a fleet of vehicles belonging to the Movhex transport company. The main objective is to minimize the travel costs of vehicles over a wide geographical area.  The map is modeled as a rectangular grid composed of hexagonal tiles. Each hexagon is uniquely identified by row and column coordinates and is connected to exactly six adjacent tiles (except for those on the borders).

## Technical and Implementation Constraints

* Language: C (C11 standard, VLA allowed)
* Libraries: Exclusive use of the standard C library, with no external libraries allowed
* Concurrency: No multithreading
* I/O: The program receives a sequence of commands via stdin and prints results to stdout
* Valuation: The implementation was tested and graded by an automated system based on strict execution time and memory limits
